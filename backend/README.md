Trivia API
-

This is a full stack webapp with API first design. Users can add questions with categories and difficulty levels. Search questions. See list of questions based on categories and play quiz games.

#### Backend
- Flask
- Flask-RESTful
- Heroku PostgreSQL 
- Docker PostgreSQL for testing


#### Frontend
- React
